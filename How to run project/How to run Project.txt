How to run Project
1. Download and Unzip the file on your local system copy Uwa .
2. Put Uwa folder inside root directory (for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

Database Configuration

Open phpmyadmin
Create Database fluxrent
Import database fluxrent.sql (available SQL File Folder inside zip package)
